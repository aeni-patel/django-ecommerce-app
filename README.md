# Django E-Commerce App

A web app for browsing products, managing a cart, and placing orders.

## Features
- User login/signup
- Product search
- Shopping cart
- Admin dashboard

## Tech
- Django, Python, Bootstrap, SQLite

## Known Issues
- Email notifications may fail
- Payment functionality currently not working

## Setup
```bash
git clone <repo_url>
cd django-ecommerce-app
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
