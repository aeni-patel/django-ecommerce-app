# Django E-Commerce App

A web app for browsing products, managing a cart, and placing orders.

## Features
- User login/signup
- Product search
- Shopping cart
- Admin dashboard
- Pyment 

## Tech
- Django, Python, Bootstrap, SQLite


## Setup
```bash
git clone <repo_url>
cd django-ecommerce-app
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
